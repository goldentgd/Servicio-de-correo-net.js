# Servicio-de-correo-net.js
Servicio de envio de correos
Ingresar el comando npm install
Crear el .env con los datos correspondientes.

PORT=0000

SMTP_HOST=smtp.gmail.com
SMTP_PORT=465
SMTP_SECURE=true

SMTP_USER="correo del que se enviará los datos"
SMTP_PASS="contraseña de aplicación"

API_TOKEN="Token creado por el usuario"
